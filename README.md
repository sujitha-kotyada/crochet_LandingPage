# 🧶 CrochetVerse – Crochet Animated Landing Page

A simple scroll-animated landing page for a crochet/handmade brand built using HTML, CSS, and JavaScript.

## ✨ Features
- Sticky navigation bar
- Hero section with headline and CTA
- Smooth scroll animations
- Fade-in and slide-up effects on scroll
- Responsive design
- Modern cozy UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## 🚀 How to Run
- Download or clone the repo
- Open `index.html` in any browser

## 💡 Purpose
This project demonstrates how AI-assisted coding can be used to build modern animated landing pages.

Built as part of a vibe coding / AI learning project.
